🤝 Office Management System (MERN)

A collaborative full-stack project built to modernize workplace attendance and employee management. This was a team effort focused on creating a seamless flow between a React frontend and a Node/MongoDB backend.

🚀 Key Features

Real-time Attendance: Quick check-in/out functionality.

History Tracking: Persistent logs of employee activity and hours.

Team Built: Developed using Git for version control and collaborative coding.

Full-Stack: Integrated MERN architecture (MongoDB, Express, React, Node).

🛠 Tech Stack

Frontend: React.js

Backend: Node.js & Express

Database: MongoDB (via Mongoose)

Collaboration: Git & GitHub

🔧 Setup

Backend: cd server ➔ npm install ➔ Setup .env ➔ npm run dev

Frontend: cd client ➔ npm install ➔ npm start

💡 Team Takeaways

Working on this as a group taught us the importance of modular code and consistent API structures. 
We focused on building a scalable system where frontend components and backend routes work in perfect sync.
